1. Sidebar (Navbar Samping Kiri)
    Menu:
            Dashboard/Beranda
            Forum/Diskusi
            Tanya/Buat Pertanyaan
            Notifikasi
            Profil
            Logout
            (Admin) Kelola User, Kelola Laporan, dsb
    Dark Mode Toggle:
            Tombol di sidebar, aktifkan dark/light mode untuk seluruh website.
    Logo:
            Pakai teks "innoforum" saja di atas sidebar, dengan font tebal/unik (bisa diganti logo gambar nanti).
    User Info:
            Foto profil kecil + nama user di bawah menu sidebar (bisa klik ke profil).

    Sidebar fixed di kiri, responsif (auto collapse jadi menu baris di bawah/atas pada HP).


2. Dashboard/Forum
    Dashboard = Forum
            Biar simpel, dashboard LANGSUNG menampilkan list pertanyaan/forum (tidak perlu statistik kecuali ingin fitur admin berupa summary).
    Akses Menu:
            Semua user (mahasiswa/dosen) dapat menu utama (di atas).
            Admin dapat menu tambahan khusus yang hanya muncul kalau role=admin.
            Tampilkan menu admin hanya jika user role=admin.
    Tombol "Buat Pertanyaan":
            Floating button di kanan bawah (modern, mudah diakses, tidak mengganggu list pertanyaan).
            Bisa juga di atas list (lebih klasik), tapi floating lebih kekinian.
            Akses: semua user (mahasiswa/dosen) bisa, admin opsional.
    Card Pertanyaan:
            Tampilkan: Foto kecil user, nama, waktu post, judul pertanyaan, isi singkat, tag/hashtag, jumlah komentar, jumlah like.
            Tombol edit/delete muncul kalau user adalah pemilik post/pertanyaan.
            Saran: Like & komentar ditampilkan berupa icon + angka, tag sebagai badge/chip kecil.
    Warna & Nuansa:
            Utama putih/abu-abu terang, aksen biru/toska untuk tombol & tag, dark mode pakai abu gelap/hitam + aksen biru/toska.
    Responsif:
            Sidebar collapse = sidebar otomatis jadi menu atas/hamburger di HP (jadi tidak makan tempat, user tinggal klik icon/menu di atas untuk buka sidebar).

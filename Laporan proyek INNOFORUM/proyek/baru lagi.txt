Step-By-Step Next
Langkah 1: Sorting/Filtering Thread (Tab)
Kita buat backend agar tab/tab filter (“Terbaru”, “Respon Terbanyak”, “Belum Dijawab”, dst) bisa bekerja.

Langkah 2: Hashtag System
Thread bisa punya banyak hashtag (pivot table).
Sidebar list hashtag, klik → tampilkan thread sesuai hashtag.
Langkah 3: Voting/Like System
User bisa like/vote pertanyaan/jawaban.
Satu user hanya bisa vote sekali per item.
Langkah 4: Reporting/Flagging
Tombol “Laporkan” di setiap thread/jawaban.
Laporan masuk ke tabel khusus untuk admin review.
Langkah 5: Statistik Forum & Top Users
Di sidebar/dashboard, tampilkan statistik forum.
Ranking user berdasarkan jumlah thread/jawaban/vote.

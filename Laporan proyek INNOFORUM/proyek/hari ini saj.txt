Penjelasan Roadmap (Bahasa Mudah & Contoh)
1. Validasi & Keamanan
Maksud: Form tidak bisa masukin script/SQL/bad input.
Contoh: Tambah validasi di controller, misal:
PHP
$request->validate(['email' => 'required|email']);
2. UI/UX
Maksud: Biar tampilan mudah digunakan dan menarik.
Contoh: Ada badge "admin", "mahasiswa" di profil, tombol loading saat submit, dark mode.
3. Statistik Forum
Maksud: Admin/user bisa lihat statistik.
Contoh:
Total pertanyaan: 120
Total komentar: 500
User ranking: "FrederiyPatria (120 likes)"
4. Diskusi Lebih Dalam
Maksud: Bisa reply ke komentar.
Contoh: Komentar A → bisa dibalas Komentar B (nested/reply).
5. Notifikasi Lebih Canggih
Maksud: Notif bisa real-time, bisa "Tandai semua dibaca".
Contoh: Ketika ada komentar/like baru, notif langsung muncul tanpa reload.
6. Search & Filter
Maksud: User bisa cari pertanyaan atau filter laporan.
Contoh: Cari pertanyaan "Laravel" atau filter laporan dengan status "pending".
7. Export/Backup
Maksud: Admin bisa backup/export data ke Excel/CSV.
Contoh: Download seluruh data pertanyaan ke file Excel.
8. Upload File/Gambar
Maksud: User bisa upload gambar/file di pertanyaan/komentar.
Contoh: Upload screenshot waktu tanya/jawab.
9. Fitur Email
Maksud: Kirim email notifikasi ke user.
Contoh: User dapat email bila ada jawaban/komentar baru.
10. Testing Otomatis
Maksud: Cek otomatis fitur berjalan benar saat update code.
Contoh: Jalankan command php artisan test untuk pastikan fitur tidak error setelah update.

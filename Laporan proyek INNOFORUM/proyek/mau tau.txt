1. UI
    - Sidebar (navbar vertikal di samping kiri):
            - Icon/logo forum di atas
            - Daftar menu/navigation (Dashboard, Forum, Tanya, Notifikasi, Profil, Logout, dsb)
            - (Kadang) ada foto/profile user di pojok bawah sidebar
    - Main Content (halaman utama):
            - Bagian atas: judul atau search bar (optional)
            - List pertanyaan/forum berupa card/list + info user, waktu, tag, jumlah komentar/like
            - Tombol "Buat Pertanyaan" di atas/bawah atau melayang di kanan bawah
            - Responsive (sidebar bisa collapse di mobile)

2. Komponen Navigasi (Sidebar) 
    Sidebar kiri (fixed/berwarna/putih):
    - Logo/brand/forum kampus (paling atas)
    - Menu utama:
        - Dashboard (atau "Beranda")
        - Forum (atau "Diskusi")
        - Tanya (link ke form buat pertanyaan)
        - Notifikasi
        - Profil
        - Logout
    - (Optional) Foto user + nama di bawah sidebar
    - (Optional) Admin bisa ada menu tambahan: "Kelola User", "Kelola Laporan", dsb
            Contoh urutan:
            Code
            [Logo]
            Dashboard
            Forum
            Tanya
            Notifikasi
            Profil
            Logout
            ----------------
            [User Info/Profile Pic]


3. Halaman Utama (Dashboard/Forum):
    - Judul halaman di atas ("Forum Kampus" / "Dashboard")
    - Tombol "Buat Pertanyaan" (atas kanan atau bawah kanan)
    - Daftar pertanyaan/forum:
        - Nama user, waktu post
        - Judul pertanyaan
        - Isi singkat/content
        - Tag/Hashtag
        - Jumlah komentar/likes
        - Tombol Edit/Delete (jika pemilik post)
        - Support responsive (sidebar collapse di mobile)

4. Progres Fitur Yang SUDAH di Buat
        Frontend:
        - Halaman validasi NIM/NIGM (user harus validasi sebelum register/login)
        - Halaman login/register (form)
        - Dashboard (masih basic)
        - Forum (list pertanyaan, sangat sederhana)
        Backend:
        - Validasi NIM/NIGM (session-based)
        - Login/logout/register user
        - CRUD User (via UserController, route resource)
        - CRUD Question (buat, edit, hapus, lihat pertanyaan)
        - CRUD Comment (buat, edit, hapus komentar)
        - CRUD Admin (route group + role middleware)
        - Middleware role
        - Session management (validasi NIM)

    Belum ada:
        - File upload (kecuali avatar user waktu register)
        - Notifikasi real-time
        - Edit profile user (hanya edit via form basic)
        - Fitur advance lain (search, filter, dsb)

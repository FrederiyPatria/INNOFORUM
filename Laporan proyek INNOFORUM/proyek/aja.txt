Rangkuman & Saran Akhir
Sidebar dengan menu utama ("innoforum" di atas), tombol dark mode, dan menu admin otomatis hanya untuk admin.
Dashboard = Forum utama: List pertanyaan, tombol "Tanya" (floating button kanan bawah).
Card pertanyaan: Nama, foto user, waktu, judul, isi singkat, tag/hashtag (dummy dulu), komentar, like.
Dark mode toggle: Di sidebar.
Aksen warna: Biru/toska (modern dan netral).
Tag/hashtag: Dummy dulu, akan diingatkan kalau sudah mau dihapus.
Responsif: Sidebar collapse otomatis di HP.
Framework: Tailwind CSS.
Struktur File yang Akan Dibuat
resources/views/layouts/app.blade.php
→ Layout master (sidebar, dark mode, slot content)
resources/views/dashboard.blade.php
→ Dashboard/forum utama (list pertanyaan, tombol tanya)
resources/views/questions/create.blade.php
→ Form buat pertanyaan (tanya)
resources/views/components/question-card.blade.php
→ (Opsional) Card pertanyaan (biar rapi/DRY)
